# Modelling the 'Dry Bean' Dataset

## About

This university project focussed on data modelling, and aimed to explore which model was more precise. The project features a python script that reads a the 'Dry Bean' dataset, cleans it and creates various visual graphs exploring its contents. 1 classification and 1 clustering model is then built, a Decision Tree model and a K-Nearest Neighbours (KNN) model. The results for each model are presented in a classification report.
- [(*Check it out on GitHub*)](https://github.com/hiba-ansari/python-dataset-analysis)

## Technologies Used
- Python
    - pandas
    - matplotlib
    - sklearn
    - numpy

## Dataset
The dataset is taken from the [UC Irvine Machine Learning Repository](https://archive.ics.uci.edu/dataset/602/dry+bean+dataset), which contains 13611 rows and 16 features.
